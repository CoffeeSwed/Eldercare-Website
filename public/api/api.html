action Parameters Notes
create_user username, password, email, contact_number, date_of_birth, address_country, address_city, address_postcode, address_street_and_house(for example: Diskusgatan 4 eller SommarGatan 116 LGH 1405),type (user type, Administrator, Caregiver, Patient or Guest) Optional: ( session & key preferences) Creates a user with specified username and password. If the user is not in a session the rules defined by the permission file (config.ini) determines if they can create the specified type of user. If the user is logged in, their user type is used to determine if they cant create the requested account by using the rules in the config.
create_session username, password Creates a session for specified username if password is correct, if success, it returns the session id and session key. If password is incorrect or the user dont exist, it returns USER_NOT_FOUND error!
is_valid_session session, key Returns true if the input is valid, otherwise false
get_session_user_info session, key Returns the current user information stored in the table users.
get_user_info session, key, optional(username, id --- index id in the users table) Returns the specified user information if requester has permission to do so - look in config.ini!
get_website_config None Returns the information that is considered public in the config file.
add_parent / delete_parent username (child username), id(child id), parent_username(parent username), parent_id(parent id), session, key Adds or removes a parent from a user.
get_meal_plan username, id, date(yyyy-mm-dd or null for today, if a meal doesn't exist for the day specified and the day is "today", then the api generates a meal plan!), session, key Gets the meal plan for specified day!
get_dinner_times_database None Returns the dinner times where a patient should eat or etc.
get_meal_database None Returns the different meal types that exists and their configurations, should be saved in cache.
get_matching_users Same input as create_user Returns user ids which has matching input as given!!
